# Demo run (real OSM map)

One-time: `python -m scripts.fetch_osm`   # writes data/hcm_drive.graphml

Run the control room on the real HCM map with a lively event stream:

    set WORLD=real
    set ENABLE_SUDDEN_EVENTS=1
    set REPLAN_INTERVAL_TICKS=4
    python -m fleet.ui.server

Open http://127.0.0.1:8000 . Default sample world (no graphml needed): just
`python -m fleet.ui.server`. The Settings gear (World = real/sample) also drives this.

## Injecting a manual test event

You don't need to wait for a random sudden event — inject one yourself from the
**Field Report** panel:

1. Type a free-text report, e.g. `Road into C001 is flooded`.
2. Click **Extract & dispatch**.

This `POST`s to `/api/report`, runs the intake pipeline (extract → resolve →
inject), and the event appears on the map like any other disruption, driving the
detector / decision / reroute path.

A click-to-inject-on-map control is a possible future addition (not built yet).
