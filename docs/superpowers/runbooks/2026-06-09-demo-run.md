# Demo run (real OSM map)

One-time: `python -m scripts.fetch_osm`   # writes data/hcm_drive.graphml

Run the control room on the real HCM map with a lively event stream:

    set WORLD=real
    set ENABLE_SUDDEN_EVENTS=1
    set REPLAN_INTERVAL_TICKS=4
    python -m fleet.ui.server

Open http://127.0.0.1:8000 . Default sample world (no graphml needed): just
`python -m fleet.ui.server`. The Settings gear (World = real/sample) also drives this.
